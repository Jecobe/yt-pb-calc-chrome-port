# Setup Development Environment 
```
pip -m venv venv/py
pip install nodeenv==1.8.0
source venv/py/bin/activate

nodeenv --node 20.6.1 --npm 9.8.1 env/node
source env/node/bin/activate

npm i rollup eslint-config-google
```

# Build Extension
```
npm run build
```
